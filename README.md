## Overview
This project included making a weather dashboard that displays portions of the current weather of a location. In addition, the user is presented with a 5 day forecast for the area. The user can search for another location and get the same information for that location.

All areas that the user searches are historically placed below the main search button.

The default location that is searched 'Minneapolis'. This value is immediately injected into the search history. This decision was made so that the user wasn't forced to add a seach term in order to understand what this website does.
